APS OCR (local)

Importez un PDF, lancez l’OCR, puis récupérez le texte et le JSON parsé (aucun envoi serveur).

1) Import

En attente Importez un PDF.

2) Aperçu & Résultats

Aperçu page (rendu)

Le PDF est rasterisé (c’est-à-dire converti en image) dans un canvas (zone de dessin HTML) avant OCR.

Exports

Le ZIP est généré côté navigateur via JSZip (bibliothèque JavaScript), sans serveur.
